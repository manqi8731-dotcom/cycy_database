# “潮韵厝影”数据库网页说明

# 一、背景与初衷
我们搭建“潮韵厝影”数据库网页，旨在提供一个轻量、便捷、无需安装的工具与服务。我们致力于宣传潮汕传统建筑文化，将数字化成果与调研所得呈现给大众，让这些珍贵的文化不再被封存，而是被更多人看见。

# 二、网页搭建过程及细节
结合我们项目的数字化特性，本网页的搭建细节如下 ：
1、技术栈：使用原生 HTML/CSS/JavaScript 构建轻量级的单页面应用（SPA）前端框架。采用 GitHub Pages 进行网页托管，利用腾讯云存储作为静态资源（图片、PDF、文档）的图床服务器。通过解析 manifest.json 清单文件实现页面的动态渲染与路由切换。针对3D模型，特别引入了谷歌官方的 <model-viewer> Web 组件以支持.glb文件的交互式预览。
2、适配设计：自适应电脑、平板、手机屏幕，保证在不同设备上的浏览体验 。
3、性能优化：通过分离存储策略提升加载速度，减少等待时间。前端代码与3D模型托管于 GitHub，而大量高清影像与文献则依托腾讯云的高速网络加载。
4、安全措施：采用 HTTPS 加密保护你的信息安全。同时移除了前端的文本编辑功能，确保展示数据的纯粹性与安全性。

# 三、网页包含内容
本数据库内容丰富，分类清晰，主要包含三大板块 ：
1、建模相关资料： 提供刘氏宗祠的3D建模文件（支持在线3D旋转放缩预览）、CAD建筑图纸（PDF下载），以及详细的部件、柱子建模截图。
2、建筑相关照片：收录了刘氏宗祠、传统民居、现代民居及空心村落的大量高清实景影像，涵盖旗杆夹、抱鼓石、嵌瓷、木雕、彩绘等潮汕建筑细节特征。
3、文献收集整理资料：提供关于潮汕建筑结构与装饰艺术的系统性文献归档，支持直接下载查阅。
不同板块分区明确，无冗余内容，用户可快速找到目标板块，无需繁琐查找。

# 四、如何使用网页
1、访问方式：复制网页网址https://manqi8731-dotcom.github.io/cycy_database/，粘贴至手机、电脑、平板的任意浏览器（如微信浏览器、Chrome、百度浏览器等），点击访问即可直接进入，无需注册、无需登录。
2、基础操作：进入网页后，首页即可看到核心板块导航，点击对应板块即可进入查看；页面支持上下滑动浏览内容，按钮、链接均有清晰标注，点击即可执行对应操作。
3、特殊说明：若页面显示异常，可刷新浏览器或清除浏览器缓存；无需担心数据丢失，网页仅做公开内容展示，不收集个人隐私信息，使用更安全。

# 五、数据与隐私
1、数据存储：本网页为纯静态展示型数据库，不设用户注册与交互输入功能。
2、隐私保护：我们不会收集非必要的个人信息。
